🎮 Quiz Game Website

This is a multiplayer quiz game where players can join a room and answer questions in real time.

🚀 How to Use
Enter your name
Create a room or join with a code
Answer questions and compete
🛠️ How to Run

Go to the server folder:
npm install
node server.js

Then open the client folder and open index.html
